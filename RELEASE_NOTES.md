# Promptly v1.0.158 - Initial Release

## 🎉 Welcome to Promptly!

This is the initial public release of **Promptly**, a professional teleprompter application designed for content creators, presenters, and professionals who need reliable, feature-rich teleprompting.

---

## ✨ Core Features

### 📝 Script Management
- **Multi-Chapter Scripts** - Organize your content into chapters with individual titles and settings
- **Rich Text Editor** - Format text with bold, italic, underline, custom colors, and adjustable sizes
- **Drag & Drop** - Reorder chapters easily
- **Import Support** - Load scripts from TXT, DOCX, and PDF files
- **Auto-Save** - Your work is automatically saved as you type

### 🎬 Professional Teleprompting
- **Dual-Window Display** - Separate control interface and presenter window
- **Smooth Scrolling** - Variable speed control (0.1x - 10x) with keyboard shortcuts
- **Custom Speed Per Chapter** - Set different scroll speeds for individual chapters
- **Adjustable Display** - Font size, color, line height, background color, and opacity controls
- **Mirror Options** - Flip horizontal and vertical for teleprompter hardware setups
- **Crosshair Guide** - Adjustable center-line guide with custom colors
- **Timer & Progress** - Real-time elapsed time, estimated duration, and progress bar

### ⌨️ Keyboard Shortcuts
- **Customizable Controls** - Assign your own keyboard shortcuts for all major functions
- **Multiple Shortcuts** - Add alternative key combinations for the same action
- **Mouse Wheel Support** - Use scroll wheel for speed adjustment
- **Default Controls:**
  - `Space` - Play/Pause
  - `↑/↓` - Adjust speed
  - `←/→` - Skip chapters
  - `Home` - Reset to beginning
  - `Escape` - Close presenter window
  - And many more...

### 📱 Remote Control
- **Phone/Tablet Control** - Control your teleprompter from any device on your network
- **QR Code Connection** - Easy setup by scanning QR code
- **Touch-Optimized Interface** - Large, easy-to-press buttons
- **Chapter Selection** - Jump directly to any chapter from your phone
- **Screen Wake Lock** - Keeps your phone screen on during use
- **Real-Time Status** - Shows when presenter window is open
- **Speed Synchronization** - Remote respects your keyboard adjustment settings

### 🎨 Customization
- **Flexible Layout** - Resizable sidebar and preview panel
- **Margin Controls** - Adjust horizontal margins and lead-in spacing
- **Chapter Spacing** - Control spacing between chapters
- **Background Opacity** - 0-100% transparency for overlay on video feeds
- **Theme** - Dark, professional interface

## 🚀 Getting Started

### Installation
1. Run `Promptly Setup 1.0.158.exe`
2. Follow the installation wizard
3. Launch Promptly from your desktop or start menu

### Quick Start Guide
1. **Create Your First Script:**
   - Type or import your content
   - Organize into chapters if needed
   - Adjust formatting to your preference

2. **Open Presenter Window:**
   - Click "Open Presenter" button
   - Position on your teleprompter screen
   - Adjust display settings as needed

3. **Start Presenting:**
   - Press `Space` to start scrolling
   - Use `↑/↓` to adjust speed on the fly
   - Press `Space` again to pause

4. **Optional - Set Up Remote Control:**
   - Click the "Remote" tab
   - Click "Start Remote Server"
   - Scan QR code with your phone
   - Control from anywhere on your network!
